CommScript GUI - Buttons

COM Port
Here you can choose the desired port. Only ports <= COM49 are listed which CommScript was able to open when the list was generated (which occurs at start-up and every time OFF is selected from the list). If opening the chosen port fails, it will automatically switch back to OFF!

Baud Rate
Here you can choose the desired baud rate. The default list entries are the most popular speeds.

Byte Size / Parity / Stop Bits
Here you can choose the additional port parameters. In the list you will find strings composed of the byte size, parity, and stop bits. Byte size may be '7' or '8' bits, parity one of 'N' (none), 'O' (odd), 'E' (even), 'M' (mark), 'S' (space), and '1' or '2' stop bits.

Flow Control
Here you can choose the flow control method. Available is 'None', 'Xon/Xoff', or 'Hardware'. (I don't know if this works... I have used only 'None' so far...)

Editing Script
You can use the 'Edit...' button to edit a CommScript script. This will invoke the IDLE editor. It is a slightly modified version from the one included with Python 2.4.3. Please see Release Notes for more details... This editor supports syntax coloring and some other nice features specifically useful for Python programs.

Running/Stopping Script
You can use the 'Run...' button to run a CommScript script. Basically, a script is a text file which contains some Python code and has the extension '.cs'. Clicking the 'Run...' button will show a file selection box where you can pick a filename. If you do, it will replace the input line with a command that you can use to run the script again, which is also appended to the input history. Also, the button text changes to 'Stop!'. You can now use the same button to stop the script again. The script must be constructed apropriately, that stopping does not stall the main window... Please see the examples in the CommScript folder and/or the section about scripts.
