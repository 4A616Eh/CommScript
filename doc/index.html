CommScript Documentation

CommScript is a scriptable serial communication tool targeted at developers. It offers many features that terminal emulator programs just don't have, like automatic time stamping of sent and received data, separate areas for input and output data, repeating inputs by selecting from a history list, editing output data (very useful to type in comments while still more data is appended to the end, or while the connected device is turned off), running Python scripts (which are able to receive and send data and even may change elements of the user interface) in a separate thread, highlighting of Perl-style regular expressions in output area, and much much more...
