CommScript Installation

Currently, CommScript is only available for Windows (and only tested on Windows XP). The installer package was created using NSIS (Nullsoft Scriptable Install System) and very easy to use! Just follow the simple instructions on the screen and CommScript will be installed in a few minutes.

After the installation is complete, you are asked if you would like to run the program... You can always run it later from the Start menu.

Very important: If you would like to run more than one CommScript window at the same time, you should make sure that the program is run from different folders. Since the program saves data and setup into the folder which was current folder when the program was started, you could overwrite files by just closing one window – which may result in loosing important information!!

Hint: Create shortcuts and modify the “Start in:” setting! For example: Click on Start, then All Programs, then right-click “CommScript” in the CommScript menu. Select “Create Shortcut”. A shortcut “CommScript (2)” will appear in the same menu. Right-click this and select “Properties”. There you will find the “Start in:” setting – change this to a different location.

Please note: running the program several times from the same folder will not give you a warning or error message. You have been warned here and now...
