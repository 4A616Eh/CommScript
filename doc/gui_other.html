CommScript GUI - Other Elements

The CommScript main window has a few more features that are worth mentioning: There is a status line at the bottom and a progress bar next to it. These two elements will be used to show the status during XModem transfers, but they are also programmable from scripts! Please see the Script Reference for further details.

The window title of the main window is also programmable and will be stored in the setup. This will help you to distinguish the windows (if you have more then one CommScript window open at the same time) in the Windows Taskbar. The command 'set_title' is also explained in the Script Reference.

Above the input line there are the checkboxes for Perl regular expression searches. Please note: you will only see those, if the window has a certain height. If you shrink the vertical size of the main window, this line will be hidden!
